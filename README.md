# Project and Task Management

**Live Demo:**  
[https://yevhen0905.github.io/project_management/]

## Overview

This project is a **Kanban-style task and project management application** built with Vue 3 and Vite.

- Task statuses: **To Do**, **In Progress**, **Done**
- Drag-and-drop tasks between columns and within columns
- Built with **Vue 3**, **Vite**, **Pinia**, **Axios**, **VueDraggable**

## Deployment Process

1. Clone the repository:

```bash
git clone https://github.com/yevhen0905/project_management.git
cd project_management
```

2. Install dependencies:

npm install

3. Run locally for development:

npm run dev

4. Build for production and deploy to GitHub Pages:

npm run deploy

Recommended Environment

IDE: VS Code with Volar
Browser: Chromium-based browsers (Chrome, Edge, Brave) or Firefox
DevTools: Vue.js Devtools for debugging Vue components

Technologies Used
Vue 3 - Frontend framework
Vite - Build tool and dev server
Pinia - State management
Axios - HTTP requests
Vue Router 4 - Client-side routing
VueDraggable - Drag-and-drop tasks
ag-grid-vue3 - table
Sass/SCSS - Styling
